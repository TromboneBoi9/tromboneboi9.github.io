This is ilo nanpa (tool of numbers), it's a calculator/timer focusing on the toki pona numbering system. It was created by jan Tolonpon.

The interface is in toki pona's hieroglyphic writing system known as sitelen pona (good writing). A chart can be found here.

pana NAdds N to the display number. weka NSubtracts N from the display number.
o alaClears the display to zero. ante nanpaThe display number is inverted, i.e. negative becomes positive and vice versa.
open tenpoStarts the timer, from the display number to zero. pini tenpoStops the timer early.
awen nanpaSaves the display number. ante sitelenChange the display between sitelen pona and Arabic digits.
weka nanpaSubtracts the saved number from the display number. pana nanpaAdds the saved number to the display number.
tu nanpaMultiplies the display number by two.

Ĉi tio estas ilo nanpa (ilo de nombroj), ĝi estas kalkulilon/tempigilon, kiu fokusita sur la Tokipona nombrosistemo. Ĝi estis kreita de jan Tolonpon.

La interfaco estas en la hieroglifa skribsistemo de Tokipono, sitelen pona (bona skribo). Diagramo troveblas ĉi tien.

pana NAdicias N al la ekranonombro. weka NSubtrahas N de la ekranonombro.
o alaForigas la ekranonombron al nulo. ante nanpaInversas la ekranonombron, t.e. negativa fariĝas pozitiva kaj inverse.
open tenpoStartas la tempigilon, de ekranonombro ĝis nulo. pini tenpoHaltigas la tempigilon frue.
awen nanpaKonservas la ekranonombro. ante sitelenAlternas la ekranon al sitelen pona aŭ arabaj ciferoj.
weka nanpaAdicias la konservitan nombron al la ekranonombro. pana nanpaSubtrahas la konservitan nombron de la akranonombro.
tu nanpaMultiplikas la ekranonombron per du.
